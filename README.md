## R_Basics_Workshop
This repository contains materials from an introductory **R Basics Workshop**, including:
- R scripts
- Jupyter or R Markdown notebooks
- Instructional PDFs and handouts

## Topics Covered

- Data types in R
- Data frames and tibbles
- Basic plotting
- Data import/export
- Packages and libraries
